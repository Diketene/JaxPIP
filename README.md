# JaxPIP

Differentiable Permutation Invariant Polynomial (PIP) descriptor implemented in
Jax.

## Features

Supported descriptors:

- Origin permutation invariant polynomial (PIP) [1,2]
- Fragment permutation invariant polynomial (Frag-PIP) [3,4]
